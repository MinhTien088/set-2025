Comparison Table of var, let, const in JavaScript

Feature var let const
Scope (Where the variable can be used) Available throughout the entire function (if declared inside a function) Available only inside the block {} where it was declared Available only inside the block {} where it was declared
Can the value be changed after declaration? ✅ Can be changed (the value inside the variable can be updated) ✅ Can be changed (the value inside the variable can be updated) ❌ Cannot be changed (once assigned, the value cannot be modified)
Can it be redeclared with the same name? ✅ Can be redeclared with the same name in the same scope ❌ Cannot be redeclared with the same name in the same scope ❌ Cannot be redeclared with the same name in the same scope

P/S: If you assigned a value to a variable without using var, let or const, it will implicitly become a global variable.
